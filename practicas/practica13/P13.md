## Ver deployment > Replica > pods 

```bash

kubectl get rs

```

## Borrar deploy 

**Se borra tanto el deploy como el pods como el replica set**

```bash
kubectl delete pod nginx-d-5d59d67564-vlqbj
kubectl get  pods 
# Ver el tiempo que lleva iniciado el que hemos borrado
```
