# Pods - Borrar

## Práctica 1: Borramos un pod

```bash
kubectl delete pod nginx1
```

## Práctica 2: Borramos un pod inmediatamente

```bash
kubectl delete pod nginx1 --now
```

## Práctica 3: Borramos todos los pods

```bash
kubectl delete pods --all
```
