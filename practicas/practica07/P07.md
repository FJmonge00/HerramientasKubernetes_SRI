# Pods - Multicontenedores

```yml
apiVersion: v1 # La versi√≥n
kind: Pod # Tipo de objeto
metadata: # Metadatos
  name: mi-app
  labels:
    app: web
spec: # Describe el estado ideal de nuestro objeto
  containers:
    - name: frontal
      image: nginx
      ports:
        - containerPort: 80
    - name: almacen
      image: redis
      ports:
        - containerPort: 6379
```

[Clic descargar YAML](multi.yml)

## Levantamos el Pod

```bash
kubectl create -f multi.yml
```
